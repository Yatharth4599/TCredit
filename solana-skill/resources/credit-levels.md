# Credit Levels — L1 through L4

Krexa has 4 credit levels. Agents start at L1 and advance by improving their Krexit Score.

## Level Comparison

| Parameter | L1 Micro | L2 Standard | L3 Growth | L4 Prime |
|-----------|----------|-------------|-----------|----------|
| Max Credit | $500 | $20,000 | $50,000 | $500,000 |
| APR | 36.50% | 29.20% | 21.90% | 18.25% |
| Daily Rate | 0.10% | 0.08% | 0.06% | 0.05% |
| Max Leverage | 1x | 2x | 3x | 5x |
| Score Requirement | None (new agent) | >= 500 | >= 650 | >= 750 |
| KYA Requirement | None | Basic | Enhanced | Full |
| Collateral Required | None | None | Optional (improves terms) | Optional (improves terms) |
| NAV Liquidation (Type A) | 90% | 85% | 80% | 80% |

## Advancing Levels

To move from one level to the next:

1. **L1 -> L2:** Achieve Krexit Score >= 500. Complete at least 1 full credit cycle (borrow and fully repay). Pass Basic KYA.
2. **L2 -> L3:** Achieve Krexit Score >= 650. Demonstrate consistent repayment over multiple cycles. Pass Enhanced KYA.
3. **L3 -> L4:** Achieve Krexit Score >= 750. Significant transaction history and lifetime volume. Pass Full KYA.

## Daily Interest Examples

| L1 | $500 | $0.50 | ~$15.00 |
| L2 | $10,000 | $8.00 | ~$240.00 |
| L3 | $50,000 | $30.00 | ~$900.00 |
| L4 | $500,000 | $250.00 | ~$7,500.00 |

## Leverage

Leverage determines how much an agent can borrow relative to its proven revenue or collateral:

- **L1 (1x):** Borrow up to $500 with no prior history
- **L2 (2x):** Borrow up to 2x monthly proven revenue
- **L3 (3x):** Borrow up to 3x monthly proven revenue
- **L4 (5x):** Borrow up to 5x monthly proven revenue

## Level Downgrade

Agents can be downgraded if their Krexit Score drops below the threshold for their current level. Downgrades trigger:
- Reduced credit limit (excess must be repaid)
- Higher APR applied to remaining balance
- Tighter liquidation thresholds
