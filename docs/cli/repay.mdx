```bash
krexa repay <amount|all>
```

Repays your credit line. Interest is paid first, then principal.

## Parameters

| Parameter | Required | Description |
|-----------|----------|-------------|
| `amount` | Yes | USDC amount or `all` for full repayment |

---

## Examples

```bash
# Repay $50
krexa repay 50

# Repay entire balance
krexa repay all
```

<Tip>
  Partial repayments still improve your score. Even small, consistent payments help your Repayment History component.
</Tip>
