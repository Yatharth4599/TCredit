## Installation

<Tabs>
  <Tab title="npx (no install)">
    ```bash
    npx @krexa/cli init
    npx @krexa/cli status
    ```
  </Tab>

  <Tab title="Global install">
    ```bash
    npm install -g @krexa/cli
    krexa init
    krexa status
    ```
  </Tab>
</Tabs>

## Quick workflow

```bash
krexa init            # One-time setup
krexa faucet          # Get test USDC (devnet)
krexa borrow 500      # Borrow $500
krexa status          # Check everything
krexa score           # Score breakdown
krexa repay 50        # Repay $50
```

## Global options

| Flag | Description |
|------|-------------|
| `--version` | Show CLI version |
| `--help` | Show help for any command |

```bash
krexa --help
krexa borrow --help
```
