# krexa faucet

Request devnet USDC tokens for testing on Solana devnet. The faucet mints test USDC (`H2SYsnzdRXrXpHpcDkedARksoxiQLGXjtAvkJg158ETP`) to your wallet.

## Usage

```bash
npx krexa faucet [options]
```

## Options

| Option | Description |
|--------|-------------|
| `--amount <number>` | Amount of USDC to request |
| `--keypair <path>` | Path to Solana keypair file |

## Example

```bash
# Request devnet USDC
npx krexa faucet

# Request a specific amount
npx krexa faucet --amount 1000
```

## API Alternative

You can also call the faucet API directly:

```bash
curl -X POST https://tcredit-backend.onrender.com/api/v1/solana/faucet/usdc \
  -H "Content-Type: application/json" \
  -d '{"wallet": "<your-wallet-address>"}'
```

## Notes

- The faucet is only available on **devnet**. It will not work on mainnet.
- Faucet USDC uses a devnet mint address and has no real value.
- Use faucet USDC to test borrowing, repaying, and LP operations before deploying to mainnet.
