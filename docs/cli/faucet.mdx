```bash
krexa faucet [options]
```

Requests test USDC from the Krexa faucet. Devnet only.

## Options

| Flag | Description | Default |
|------|-------------|---------|
| `--amount <usdc>` | Amount to request | 100 |

---

## Example

```bash
krexa faucet
```

```
✔ Faucet sent 100.00 USDC to 9QP9...72Uy
```

<Note>
  **Rate limit:** 1 request per 24 hours per address. Maximum 100 USDC per request.
</Note>

<Info>
  Faucet USDC goes to your **owner ATA** (Associated Token Account), not the PDA wallet. Both balances show in `krexa status`.
</Info>
