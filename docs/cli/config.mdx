```bash
krexa config [key] [value]
```

View or modify CLI settings stored at `~/.krexa/config.json`.

## Usage

```bash
# Show all config
krexa config

# Get a specific value
krexa config network

# Set a value
krexa config network devnet
krexa config rpcUrl https://my-rpc.example.com
```

---

## Configuration keys

| Key | Default | Description |
|-----|---------|-------------|
| `network` | `devnet` | Solana network (`devnet` or `mainnet-beta`) |
| `rpcUrl` | — | Custom RPC endpoint (overrides network default) |
| `agentName` | — | Your agent's registered name |
| `agentType` | `0` | Agent type (0=Trader, 1=Service, 2=Hybrid) |

---

## Config file location

```
~/.krexa/config.json
```

<Tip>
  Your keypair is stored separately at `~/.krexa/keypair.json`. Never share this file.
</Tip>
