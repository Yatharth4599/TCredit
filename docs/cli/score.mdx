---
title: krexa score
description: Krexit Score breakdown by component.
---

Shows your score with a breakdown of all 5 components.

## Usage

```bash
krexa score [wallet]
```

## Arguments

| Argument | Required | Description |
|----------|----------|------------|
| `wallet` | No | Check another agent's score by public key |

## Example

```bash
krexa score
```
