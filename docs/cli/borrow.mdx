```bash
krexa borrow <amount>
```

Requests a credit line from the Krexa vault. The oracle verifies your score and co-signs the transaction.

## Parameters

| Parameter | Required | Description |
|-----------|----------|-------------|
| `amount` | Yes | Amount in USDC (e.g., `500`) |

---

## Credit limits

| Level | Max Borrow | APR |
|-------|-----------|-----|
| L1 Micro | $500 | 36.50% |
| L2 Standard | $20,000 | 29.20% |
| L3 Growth | $50,000 | 21.90% |
| L4 Prime | $500,000 | 18.25% |

---

## Example

```bash
krexa borrow 500
```

```
✔ Oracle approved credit request
✔ Credit line opened: $500.00 USDC

Terms:
  Amount       $500.00 USDC
  Rate         36.50% APR
  Daily Cost   $0.50
  Level        L1 (Micro)
```

<Warning>
  You must run `krexa init` first. The oracle will reject requests from unregistered agents.
</Warning>
