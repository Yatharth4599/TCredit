```bash
krexa lp <command>
```

Manage liquidity provider positions across the three-tranche vault.

## Subcommands

### Deposit

```bash
krexa lp deposit <amount> --tranche <senior|mezzanine|junior>
```

### Withdraw

```bash
krexa lp withdraw <amount|all> --tranche <senior|mezzanine|junior>
```

### Status

```bash
krexa lp status
```

```

  Junior      $500.00    (20.00% APR)

```

---

## Tranches

| Tranche | APR | Risk | Loss Priority |
|---------|-----|------|--------------|
| Senior | 10% | Low | Last |
| Mezzanine | 12% | Medium | Second |
| Junior | 20% | High | First |

<Warning>
  LP deposits carry risk. Agent defaults are absorbed by Junior first, then Mezzanine, then Senior.
</Warning>
