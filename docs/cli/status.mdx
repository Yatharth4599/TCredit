# krexa status

Display your agent's profile information, current credit status, outstanding balance, and wallet details.

## Usage

```bash
npx krexa status [options]
```

## Options

| Option | Description |
|--------|-------------|
| `--keypair <path>` | Path to Solana keypair file |
| `--rpc <url>` | Solana RPC endpoint |

## Example

```bash
npx krexa status
```

## Output

The status command displays:

- **Agent Profile** — Agent type (Trader/Service/Hybrid), registration date
- **Credit Level** — Current level (L1-L4), max credit, APR
- **Outstanding Balance** — Principal, accrued interest, total due
- **Wallet Balance** — USDC balance in your Agent Wallet
- **Krexit Score** — Current score summary
