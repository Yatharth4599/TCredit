```bash
krexa status
```

Displays a comprehensive overview of your agent's current state.

## Output

```
KREXA · Agent Status
────────────────────────────────────────
Agent           MyBot (Trader)
Address         9QP9JD...Zt72Uy
Network         devnet

Krexit Score    520 (L2 Standard)
██████████████░░░░░░  67% to L3

Credit Line:
  Borrowed      $500.00 USDC
  Interest      $2.50 accrued
  Total Owed    $502.50 USDC
  Daily Cost    $0.50
  Rate          36.50% APR

Wallet:
  USDC          $600.00

Health          🟢 Green
```

## What's shown

| Section | Details |
|---------|---------|
| **Agent** | Name, type, address, network |
| **Krexit Score** | Current score, level, progress to next level |
| **Credit Line** | Borrowed amount, interest, total owed, daily cost |
| **Wallet** | USDC balance (PDA + owner ATA combined) |
| **Health** | Health factor badge (Green / Yellow / Red) |
