```bash
krexa wallet
```

Shows your SOL balance, PDA wallet USDC, and owner ATA USDC.

## Output

```
KREXA · Wallet
────────────────────────────────────────
Address         9QP9JD...Zt72Uy
SOL             1.85 SOL
PDA USDC        $500.00
Owner USDC      $100.00
Total USDC      $600.00
```

<Info>
  **PDA USDC** is held in your program-controlled wallet (subject to Revenue Router rules). **Owner USDC** is in your standard token account.
</Info>
