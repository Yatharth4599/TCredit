---
title: krexa wallet
description: PDA wallet balance and details.
---

Shows your SOL balance, PDA wallet USDC, and owner ATA USDC.

## Usage

```bash
krexa wallet
```

## Example

```
  KREXA · Wallet
  ────────────────────────────────────────
  Address         7xK2mN...3pF4
  SOL             1.85 SOL
  PDA USDC        $500.00
  Owner USDC      $100.00
  Total USDC      $600.00
```

**PDA USDC** is in your program-controlled wallet (subject to Revenue Router). **Owner USDC** is in your standard token account.
