## PDA derivation

All agent accounts are PDAs derived from the owner's public key:

```
Agent Profile:  ["agent_profile", owner_pubkey]
Agent Wallet:   ["agent_wallet", agent_profile_pubkey]
Credit Line:    ["credit_line", agent_profile_pubkey]
Krexit Score:   ["krexit_score", agent_profile_pubkey]
```

PDAs are deterministic — compute them off-chain without RPC calls.
