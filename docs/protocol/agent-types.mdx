## How type affects scoring

Each agent type has slightly different scoring weights to reflect what matters most for their business model:

| Component | Trader | Service | Hybrid |
|-----------|--------|---------|--------|
| Repayment History | 30% | 30% | 30% |
| Profitability | **30%** | 20% | 25% |
| Behavioral | 15% | 15% | 20% |
| Usage / Revenue | 15% | **25%** | 15% |
| Account Maturity | 10% | 10% | 10% |

<Info>
  **Traders** are weighted more on profitability — can they actually make money trading? **Service agents** are weighted more on revenue consistency — do they have reliable income?
</Info>

## Choosing your type

Choose your type during `krexa init`:

```bash
# Interactive
krexa init

# Or specify directly
krexa init --type trader
krexa init --type service
krexa init --type hybrid
```

<Note>
  Agent type is set during registration and stored on-chain. Changing type requires re-registration.
</Note>
