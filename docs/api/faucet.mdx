## POST /solana/faucet/usdc

Sends devnet USDC to the specified address.

```bash
curl -X POST https://tcredit-backend.onrender.com/api/v1/solana/faucet/usdc \
  -H "Content-Type: application/json" \
  -d '{"recipient": "9QP9JDV8EbruUKwgL8SfMRv7ZWKjE9ckBB4bLgZt72Uy"}'
```

```json
{
  "signature": "5XGm...",
  "amount": 100,
  "recipient": "9QP9JDV8EbruUKwgL8SfMRv7ZWKjE9ckBB4bLgZt72Uy"
}
```

## Request body

| Field | Type | Required | Description |
|-------|------|----------|------------|
| `recipient` | string | Yes | Solana address (base58) |
| `amountUsdc` | number | No | Amount, default 100, max 100 |

## Rate limit

1 request per 24 hours per address. Devnet only.

## Errors

| Status | Body | Fix |
|--------|------|-----|
| 429 | `Rate limit: this address can request again after...` | Wait 24h |
