## Request test USDC

```
POST /solana/faucet/usdc
```

Sends devnet USDC to the specified recipient address.

### Request body

```json
{
  "recipient": "9QP9JDV8EbruUKwgL8SfMRv7ZWKjE9ckBB4bLgZt72Uy",
  "amountUsdc": 100
}
```

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `recipient` | string | Yes | Solana public key (base58) |
| `amountUsdc` | number | No | Amount in USDC (default: 100, max: 100) |

### Response

```json
{
  "signature": "5XGm...",
  "amount": 100,
  "recipient": "9QP9JDV8EbruUKwgL8SfMRv7ZWKjE9ckBB4bLgZt72Uy"
}
```

### Example

```bash
curl -X POST https://tcredit-backend.onrender.com/api/v1/solana/faucet/usdc \
  -H "Content-Type: application/json" \
  -d '{"recipient": "9QP9JDV8EbruUKwgL8SfMRv7ZWKjE9ckBB4bLgZt72Uy"}'
```

<Warning>
  **Rate limit:** 1 request per 24 hours per address. Devnet only.
</Warning>
