# Vault API

Retrieve statistics about the Krexa Credit Vault, including total value locked, tranche balances, and utilization rates.

## Get Vault Statistics

```
GET /solana/vault/stats
```

### Example Request

```bash
curl https://tcredit-backend.onrender.com/api/v1/solana/vault/stats
```

### Example Response

```json
{
  "success": true,
  "data": {
    "totalValueLocked": 1250000,
    "totalBorrowed": 875000,
    "utilizationRate": 0.70,
    "tranches": {
      "senior": {
        "balance": 600000,
        "targetApr": 10.0,
        "depositors": 45
      },
      "mezzanine": {
        "balance": 350000,
        "targetApr": 12.0,
        "depositors": 28
      },
      "junior": {
        "balance": 300000,
        "targetApr": 20.0,
        "depositors": 15
      }
    },
    "activeLoans": 120,
    "defaultRate": 0.02
  }
}
```

### Response Fields

| Field | Type | Description |
|-------|------|-------------|
| `totalValueLocked` | number | Total USDC deposited across all tranches |
| `totalBorrowed` | number | Total USDC currently lent to agents |
| `utilizationRate` | number | Ratio of borrowed to deposited (0-1) |
| `tranches` | object | Per-tranche balance, APR, and depositor count |
| `activeLoans` | number | Number of agents with outstanding balances |
| `defaultRate` | number | Historical default rate |

### Notes

- This endpoint is public and requires no authentication.
- Statistics are updated in near real-time based on on-chain data.
- Use `npx krexa lp stats` for a formatted CLI view of the same data.
