title: Vault API
description: Query vault TVL, utilization, and tranche stats.
---

## GET /solana/vault/stats

Returns vault statistics.

```bash
curl https://tcredit-backend.onrender.com/api/v1/solana/vault/stats
```

```json
{
  "tvl": 1500000,
  "totalBorrowed": 750000,
  "utilizationRate": 0.50,
  "tranches": {
    "senior": { "deposits": 800000, "apr": 10.00, "utilization": 0.45 },
    "mezzanine": { "deposits": 400000, "apr": 12.00, "utilization": 0.55 },
    "junior": { "deposits": 300000, "apr": 20.00, "utilization": 0.65 }
  }
}
```

## Response fields

| Field | Type | Description |
|-------|------|------------|
| `tvl` | number | Total value locked (USDC, 6 decimals) |
| `totalBorrowed` | number | USDC currently lent out |
| `utilizationRate` | number | Ratio of borrowed to TVL (0-1) |
| `tranches` | object | Per-tranche deposits, APR, utilization |

Public endpoint. No authentication.
