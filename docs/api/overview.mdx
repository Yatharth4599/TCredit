---
title: API Overview
description: REST API for scores, credit, faucet, and vault stats.
---

Base URL:

```
https://tcredit-backend.onrender.com/api/v1
```

## Endpoints

| Method | Path | Description |
|--------|------|------------|
| GET | `/solana/score/:agent` | Krexit Score and components |
| GET | `/solana/credit/:agent/eligibility` | Credit eligibility check |
| GET | `/solana/credit/:agent/line` | Active credit line details |
| GET | `/solana/vault/stats` | Vault TVL and tranche data |
| POST | `/solana/faucet/usdc` | Request devnet USDC |
| POST | `/solana/oracle/sign-credit` | Oracle co-sign a borrow tx |

## Authentication

Most endpoints are public. Oracle signing requires a valid agent registration.

## Errors

```json
{
  "error": "AppError",
  "message": "Description of the error"
}
```

| Status | Meaning |
|--------|---------|
| 400 | Bad request / validation failed |
| 404 | Agent or resource not found |
| 429 | Rate limited |
| 500 | Server error |

<Note>
The backend runs on Render free tier. First request after inactivity may take 30-60 seconds.
</Note>
