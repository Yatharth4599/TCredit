
## Authentication

Most endpoints are **public** and don't require authentication. The oracle endpoint requires a valid agent signature.

---

## Response format

All responses return JSON:

```json
{
  "data": { ... },
  "error": null
}
```

Error responses:

```json
{
  "error": "AppError",
  "message": "Description of the error"
}
```
