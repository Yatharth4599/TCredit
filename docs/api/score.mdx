# Score API

Retrieve an agent's Krexit Score and component breakdown.

## Get Agent Score

```
GET /solana/score/:agent
```

### Parameters

| Parameter | Type | Location | Description |
|-----------|------|----------|-------------|
| `agent` | string | Path | Agent's Solana public key |

### Example Request

```bash
curl https://tcredit-backend.onrender.com/api/v1/solana/score/7xKpR9...3mN2
```

### Example Response

```json
{
  "success": true,
  "data": {
    "agent": "7xKpR9...3mN2",
    "score": 542,
    "components": {
      "repaymentHistory": 72,
      "profitability": 58,
      "behavioral": 45,
      "usagePatterns": 50,
      "accountMaturity": 30
    },
    "creditLevel": "L2",
    "maxCredit": 20000,
    "apr": 29.20
  }
}
```

### Response Fields

| Field | Type | Description |
|-------|------|-------------|
| `score` | number | Overall Krexit Score (0-1000) |
| `components` | object | Individual component scores (0-100 each) |
| `creditLevel` | string | Current credit level (L1-L4) |
| `maxCredit` | number | Maximum credit in USDC |
| `apr` | number | Annual percentage rate |

### Errors

| Status | Description |
|--------|-------------|
| 404 | Agent not found — the provided public key has no registered profile |
