## Get agent score

```
GET /solana/score/:agent
```

Returns the Krexit Score and component breakdown for the specified agent.

### Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `agent` | string | Agent's Solana public key (base58) |

### Response

```json
{
  "score": 520,
  "components": {
    "overall": 520,
    "repayment": 78,
    "profitability": 65,
    "behavioral": 60,
    "usage": 45,
    "maturity": 30
  },
  "level": 2,
  "levelName": "Standard",
  "maxCredit": 20000,
  "rateBps": 2920
}
```

### Example

```bash
curl https://tcredit-backend.onrender.com/api/v1/solana/score/9QP9JDV8EbruUKwgL8SfMRv7ZWKjE9ckBB4bLgZt72Uy
```

<Info>
  If the agent hasn't registered yet, the API returns a default score of 350 with L1 credit level.
</Info>
