---
title: Score API
description: Query Krexit Score for any wallet.
---

## GET /solana/score/:agent

Returns the Krexit Score and component breakdown.

```bash
curl https://tcredit-backend.onrender.com/api/v1/solana/score/9QP9JDV8EbruUKwgL8SfMRv7ZWKjE9ckBB4bLgZt72Uy
```

```json
{
  "source": "preview",
  "agentPubkey": "9QP9JDV8EbruUKwgL8SfMRv7ZWKjE9ckBB4bLgZt72Uy",
  "isRegistered": false,
  "preview": {
    "score": 402,
    "network": "devnet",
    "txCount": 1,
    "breakdown": {
      "base": 200,
      "walletAge": 0,
      "transactionActivity": 137
    },
    "components": {
      "c1Repayment": 7000,
      "c2Profitability": 0,
      "c3Behavioral": 5000,
      "c4Usage": 0,
      "c5Maturity": 16
    }
  }
}
```

## Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|------------|
| `agent` | string (path) | Yes | Solana wallet address (base58) |

## Response fields

| Field | Type | Description |
|-------|------|------------|
| `source` | string | `"onchain"` or `"preview"` |
| `isRegistered` | boolean | Whether agent has a Krexa profile |
| `preview.score` | number | Krexit Score (200-850) |
| `preview.components` | object | Raw component scores (basis points) |

Unregistered wallets get a preview score based on wallet age, transaction count, and SOL balance.
