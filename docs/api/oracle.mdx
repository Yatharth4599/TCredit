## Check eligibility

```
GET /solana/credit/:agent/eligibility
```

Check if an agent is eligible for credit and their current limits.

### Response

```json
{
  "eligible": true,
  "creditLevel": 2,
  "maxCredit": 20000,
  "currentDebt": 500,
  "availableCredit": 19500,
  "rateBps": 2920
}
```

## Co-sign credit transaction

```
POST /solana/oracle/sign-credit
```

The oracle verifies the agent's score, checks vault liquidity, and returns a partially-signed Solana transaction.

### Request body

```json
{
  "agentPubkey": "9QP9JDV8EbruUKwgL8SfMRv7ZWKjE9ckBB4bLgZt72Uy",
  "agentOrOwnerPubkey": "9QP9JDV8EbruUKwgL8SfMRv7ZWKjE9ckBB4bLgZt72Uy",
  "amount": "500000000",
  "rateBps": 3650,
  "creditLevel": 1,
  "collateralValueUsdc": "0"
}
```

| Field | Type | Description |
|-------|------|-------------|
| `agentPubkey` | string | Agent's public key |
| `agentOrOwnerPubkey` | string | Signer's public key |
| `amount` | string | Amount in USDC lamports (string of digits) |
| `rateBps` | number | Interest rate in basis points |
| `creditLevel` | number | Agent's credit level (1-4) |
| `collateralValueUsdc` | string | Collateral value (usually "0") |

<Warning>
  The `amount` field must be a **string of digits** (e.g., `"500000000"` for 500 USDC). Sending a number will fail validation.
</Warning>

### Response

```json
{
  "transaction": "base64-encoded-partially-signed-transaction",
  "message": "Credit approved"
}
```

<Info>
  The returned transaction is **partially signed** by the oracle. The agent must add their signature before submitting to Solana.
</Info>
